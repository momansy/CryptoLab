# CryptoLab

Interactive cryptography learning toolkit built with React, Vite, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Then open the local link shown in the terminal.

## Build

```bash
npm run build
```

## Deploy on GitHub Pages

1. Create a GitHub repository.
2. Upload/push all files in this folder.
3. Go to repository Settings > Pages.
4. Choose Source: GitHub Actions.
5. Push to the `main` branch.
6. Open the deployed link after the workflow finishes.

The project uses `base: './'` in `vite.config.js`, so it works even if the repository name changes.
